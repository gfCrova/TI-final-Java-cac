# App CRUD de Productos
###  Proyecto Final CaC

<p>
Proyecto Java con arquetipo de aplicación web Maven. Se ejecuta en el servidor Tomcat. Utiliza MySQL para la base de datos.

Esta programa te permite:
- ✔ Dar de alta un producto
- 👁‍🗨 Ver/Listar
- 🔍 Buscar
- 💫 Editar/Actualizar
- ❌ Eliminar
</p>
